# أبابيل غزة V2.0 - بلوتوث متعدد اللاعبين

لعبة قتال جوية - طوفان السماء

## البناء السحابي (Cloud Build)

1. ارفع هذا المستودع إلى GitHub
2. اذهب إلى **Actions** tab
3. اضغط **"Build APK"** → **"Run workflow"**
4. انتظر 5-7 دقائق
5. نزل الملف من **Artifacts** (app-debug.apk)

أو ادفع أي commit لفرع main ويشتغل تلقائياً.

## الميزات الجديدة في V2.0
